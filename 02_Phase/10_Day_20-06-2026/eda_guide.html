📦 Step 14: Outlier Detection & Boxplot
Step 14
Hinglish Samjho: "Unusual extreme values (outliers) ko target karo jo average results ko kharab karti hain, using Boxplot diagram."
Boxplot Anatomy: Box represents 50% middle values (Q1 to Q3). The Interquartile Range is IQR = Q3 - Q1. Limits are calculated as Q3 + 1.5*IQR and Q1 - 1.5*IQR. Points lying outside limits (red circles) are detected as outliers. 
# Calculate IQR
Q1 = df["Age"].quantile(0.25)
Q3 = df["Age"].quantile(0.75)
IQR = Q3 - Q1
lower = Q1 - 1.5 * IQR
upper = Q3 + 1.5 * IQR

# Remove outliers from Age
df_cleaned = df[(df["Age"] >= lower) & (df["Age"] <= upper)]
print(f"Age boundaries: {lower} to {upper}")
Age boundaries: -6.0 to 66.0 Total outliers removed: 2
📈 Data Visualization Intro: Why & How?
Viz Intro
Hinglish Samjho: "Data ko numbers me padhne ke bajaye charts me dekhna taaki human brain patterns aur trends ko ek second me samajh sake. Isi ko visualization kehte hain."

Visualization kyon zaroori hai?

  • Raw tables (jaise 10,000 rows CSV) me se immediate insight nikalna impossible hai.
  • Outliers, correlation trends, aur distribution clusters visually ek glance me spot ho jate hain.

Standard Matplotlib Workflow

  • Step 1: Plot area scale karo: plt.figure(figsize=(10, 5))
  • Step 2: Data plot karo: plt.plot() or plt.bar()
  • Step 3: Details labels set karo: plt.title(), plt.xlabel()
  • Step 4: Screen check grids set karo: plt.grid(True)
  • Step 5: Plot draw show karo: plt.show()
📈 Line Plot (Trends over continuous values)
Viz Step 1
Hinglish Samjho: "Time aur continuous points ke saath variables kaise change ho rahe hain (e.g. sales trends, temperature change over months) use trace karne ke liye."
Line Plot: Graph represent linear progress showing gradual increase in values over months. 
import matplotlib.pyplot as plt

# Custom Data
months = ["Jan", "Feb", "Mar", "Apr", "May"]
sales = [120, 150, 140, 190, 210]

plt.figure(figsize=(8, 4))
plt.plot(months, sales, color="blue", marker="o", linestyle="--", linewidth=2)
plt.title("Monthly Sales Trends")
plt.xlabel("Month")
plt.ylabel("Sales ($)")
plt.grid(True, linestyle=":", alpha=0.6)
plt.show()
📊 Bar Chart (Categorical Comparison)
Viz Step 2
Hinglish Samjho: "Different groups ya categories ko compare karne ke liye. (E.g. Males kitne the aur Females kitni thin)."
Bar Chart Comparison: Showing absolute distribution metrics across Class 1, 2, and 3. 
# Survival by Passenger Class comparison using pandas plot wrapper
class_counts = df["Pclass"].value_counts()

plt.figure(figsize=(6, 4))
plt.bar(class_counts.index.astype(str), class_counts.values, color=["red", "blue", "green"], width=0.6)
plt.title("Passenger Class Counts")
plt.xlabel("Passenger Class")
plt.ylabel("Count")
plt.grid(True, axis="y", linestyle="--", alpha=0.5)
plt.show()

📝 Matplotlib Customization Cheat Sheet
Viz Help
Hinglish Samjho: "Apne graph ko sundar, readable aur informative banane ke properties parameters." 
# 1. Size control:
plt.figure(figsize=(width_inches, height_inches))

# 2. Main Title:
plt.title("Main Heading of the Chart", fontsize=16, fontweight="bold", color="darkblue")

# 3. Axis Labels:
plt.xlabel("Name of X-Axis (e.g. Category)")
plt.ylabel("Name of Y-Axis (e.g. Values)")

# 4. Show Grid:
plt.grid(True, linestyle="--", alpha=0.5) # alpha defines transparency of grid lines

# 5. Rotate Category Labels on X-axis (useful for long names):
plt.xticks(rotation=45)

# 6. Legends display (indicators mapping):
plt.legend(["Died", "Survived"], loc="upper right")
